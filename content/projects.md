## irishbuoys

**Website:** [johngavin.github.io/irishbuoys](https://johngavin.github.io/irishbuoys/)

- Real-time Irish Weather Buoy Network data from Marine Institute ERDDAP
- Interactive dashboards with wave height, wind speed, and rogue wave detection
- DuckDB-backed data pipeline with quality validation

- [Static Dashboard](https://johngavin.github.io/irishbuoys/articles/dashboard_static.html)

## footbet

**Repository:** [github.com/JohnGavin/footbet](https://github.com/JohnGavin/footbet)

- Football match prediction models (Poisson GLM, Dixon-Coles, Bayesian brms)
- Data ingestion from football-data.co.uk with robust parsing
- Expected Goals (xG) features via FBref/worldfootballR
- Kelly criterion betting simulation and value detection

Tags: R, statistics, sports-analytics, Bayesian
